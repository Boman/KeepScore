  • Added ICS Holo theming, action bar, and new home screen.
  • Better German translations by Werner H.

  • Small encoding fix for ICS.

  • Enabled multi-delete.
1.1.1
  • Added "show round totals" option.
  • Brought back the reassuring "game saved" message.
  • Substantial performance improvements. Games should load more quickly now.
1.1
  • Fixed bug where scores might rarely be lost. Saving is now fast, automatic, and invisible to the user.
  • Player names are autosuggested. (Thanks to Dustin Mitchell for coding this!)
  • De-uglified much of the app, changed green text to blue for colorblind folks, made green/blue configurable.
  • Added "Copy Game" function.
  • Added "Shuffle" function. (Thanks, Rob!)
  • Pop-up buttons and on-screen buttons are managed independently. (Thanks to Staffan Tylen for the suggestion.)
  • On-screen buttons (-10, -5, +5, +10) always shown for tablets.
  • Use Proguard for smaller APK size.
  • UI simplifications and performance improvements.
